# Booking App

A sleek and modern Flutter-based booking application designed with smooth navigation, elegant UI, and Firebase-powered authentication.

---

## Features

- Authentication with google

- State Management

- Save data in local DB
---
## Tech Stack

- Flutter

- Riverpod

- Firebase Auth

- Cloud Firestore

- Sqflite
